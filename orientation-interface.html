Introduction

We'll point out some of the interface elements referred to in subsequent sections of this guide. At the end check out the Try This section for exercises.

MiniMap ( see your code from 10,000 feet )

Sublime uses hardware rendering to create a scrollable zoomed out perspective of your code. Along with multiple selections, the mini-map enables an innovative new way of making multiple changes to text at once.

QuickPanel

The QuickPanel is an overlay list menu. As you type in words any items not containing them are filtered from the list. This quick interface is used for opening projects file and swapping between open files. It it also possible to use it with plugin commands.

  • Negative search can be achieved by prefixing a search token with a - ala -money
  • When in file mode the path section of a file is visually indicated with a faded grey color. Search terms will not be matched unless a slash is included as part of the token

Console

The console is where you can see all logging and can interact with Sublime by inputting Python code.

Try this

QuickPanel

  1. Open (or create) a few files
  2. Press ctrl+shift+o to open the selectFile command which uses QuickPanel for input
  3. Swap between files
  4. Try a negative query by prefixing a -
  5. Target a path section of a list item by including a slash in one of the search terms
  6. Upper case letters are handled as abbreviations. MS will match multipleSelection

Console

  1. Toggle the Console with ctrl+backquote and watch the output as you save a file

MiniMap

  1. Toggle the MiniMap with alt+m and scroll around the document
  2. Move your cursor about and look to see it moving in the Mini Map
