Introduction

All operations in Sublime are implemented via commands. There are commands related to each of the main objects in the application hierarchy except for projects.

No commands are hardcoded to any particular interface, be it keybinding or menu item.

There are many internal core commands and they can also be created with Python plugins.

Try this

  • Press ctrl and backquote to open the Python console
  • Try running some view and window commands via the Python API
  • view.runCommand(cmd) or window.runCommand(cmd)
  • Notice that a view can't run a window command such as new

Bonus Points

  • import sublime
  • sublime.runCommand('newWindow')

Python API WTF!?

All you need to know at this point is that there is a difference between commands that operate on the window and commands that work on the view. There are also application level commands such as newWindow
