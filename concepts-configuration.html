Introduction

Sublime is extremely configurable. As mentioned in the command section, nothing is hard coded. Every keybinding is customizable as are all of the drop down menus. Every thing is configured by editing plain text files, some structured in xml, others in a very simple ini like format.

We won't get into too much of the nitty gritty as this Getting Started is supposed to serve as a high level overview without going into too much detail. References will be linked to at the end and we'll walk through setting your favorite theme / font soon enough.

Opening Warning

It is HIGHLY recommended to put your data directory under version control. Commit before every update of Sublime. At the time of writing the Package upgrade rules don't account for user modifications to packages that ship with Sublime. eg changes to assets in the Python package that comes with Sublime.

It is therefore possible, however unlikely, to (silently) lose your customized settings inbetween updates. This is far from ideal but not really that big of a deal as updates aren't that frequent.

There are some best practices you can follow to minimize the risk. Some of the settings can be placed in a User package which is automatically merged with defaults but in some cases it's impossible to avoid having to manually merge in changes.

WTF!? That sounds tedious

Pretty much anyone using a text editor should be using version control and as such most will find the experience relatively painless.

A future version of Sublime may include a GUI ( a sad prospect ) for configuration / customization to make automated merging possible. Until then practice safe configuration and enjoy the great flexibility Sublime has to offer.

Now that's out of the way let's look at what is customizable and where

What?

  • keybindings
  • fonts
  • themes
  • drop down menus
  • context menus

Where?

Options (*.sublime-option)

These files are plain text ini style.

Key Bindings (*.sublime-keymap)

These are all configured in XML

Menus (*.package-menu, *.sublime-menu)

These are all configured in XML

Try this

  • Open the python console
  • Create a new view (tab)
  • Type in and evaluate the following python
  • # Set the rulers
view.options().set('rulers', 80)

# Print the current theme in use
print view.options().get('colorscheme')

# Print the current syntax in use
print view.options().get('syntax')

# Print the current font in use
view.options().get('font')
